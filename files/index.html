Features

Combine in a single tool access to several Redis servers where the app can access to.

It can be run locally or in a server, maybe just for the time you need to debug keys.

Easy to use

The scripts combines libraries to access Redis and an HTTP server to serve the UI through your preferred broswer.

Independence

Don't need to install anything else, so you can scp the tool to your server and run it when you need it, independently of libraries or the OS you have installed.
